# Rails API template

## Stack & Features

* Ruby 2.5.3
* Rails 5.2.1.1
* API mode
* Postgres
* Puma
* Handling CORS
* Fast JSON:API serializer for Ruby Objects
* RSpec testing framework
* Configurable and extendable Git hook
